ZeroAuth SDK

Passwordless ZK credential verification

Demo 1: React Button Component
Demo 2: Vanilla JavaScript

Quick Start

1. Configure (before importing)
// Configure ZeroAuth globally
window.__ZERO_AUTH_CONFIG__ = {
  relayUrl: 'https://your-relay-server.com',
  apiKey: 'your-api-key',  // Optional
  verifierName: 'Your App Name',
  timeout: 60
};
2. React Usage
// Using the Button Component
import { ZeroAuthButton } from '@zero-auth/sdk';

<ZeroAuthButton
  request={{ credentialType: 'Age Verification' }}
  onSuccess={(result) => console.log(result)}
  onError={(error) => console.error(error)}
/>
3. Vanilla JavaScript Usage
// Using the Class API
import { ZeroAuth } from '@zero-auth/sdk';

const zeroAuth = new ZeroAuth({
  relayUrl: 'https://your-relay.com'
});

const result = await zeroAuth.verify({
  credentialType: 'Age Verification',
  claims: ['birth_year']
});

if (result.success) {
  console.log('Verified!', result.claims);
}
4. Generate QR Code
// Generate QR as base64 image
const qrDataUrl = await zeroAuth.generateQRBase64(
  qrPayload, 
  { width: 256, color: '#000000' }
);

// Or generate deep link for wallet
const deepLink = zeroAuth.generateDeeplink(sessionId);
// Result: zeroauth://verify?session=abc123
